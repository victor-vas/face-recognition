## :information_source: How To Use

```bash
# Clone this repository
$ git clone https://github.com/victor-vas/face-recognition-app face-recognition-app

# Go into the repository
$ cd face-recognition-app

# Install dependencies
$ yarn

# Run the app
$ yarn start
```
